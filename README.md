# 🏋️ Gym RAG Chatbot

A fully local **Retrieval-Augmented Generation (RAG)** chatbot for Gym & Fitness questions — powered by FAISS, sentence-transformers, TinyLlama, and FastAPI. No paid APIs required.

---

## 🧰 Tech Stack

| Layer           | Technology                               |
|-----------------|------------------------------------------|
| API Framework   | FastAPI + Uvicorn                        |
| Embedding Model | `sentence-transformers/all-MiniLM-L6-v2` |
| Vector Database | FAISS (faiss-cpu) — cosine similarity    |
| Language Model  | `TinyLlama/TinyLlama-1.1B-Chat-v1.0`    |
| Dataset         | Custom JSON (25 gym Q&A entries)         |
| Language        | Python 3.10–3.12 (3.13 not supported)     |

> ⚠️ **Note:** Python 3.13 is not yet supported by key dependencies (NumPy, PyTorch, FAISS). Use Python 3.12 or 3.11 for best compatibility.

## 🔄 How RAG Works in This Project

```
User Question
     │
     ▼
┌─────────────────────────────┐
│  Embedding (all-MiniLM-L6)  │  ← Convert question to 384-dim vector
└────────────┬────────────────┘
             │
             ▼
┌─────────────────────────────┐
│  FAISS Vector Search        │  ← Cosine similarity search
│  (IndexFlatIP, normalised)  │  ← Returns Top-3 nearest documents
└────────────┬────────────────┘
             │
             ▼
┌─────────────────────────────┐
│  Context Builder            │  ← Formats retrieved Q&A into a prompt
└────────────┬────────────────┘
             │
             ▼
┌─────────────────────────────┐
│  TinyLlama-1.1B-Chat        │  ← Local LLM generates the final answer
│  (HuggingFace Transformers) │
└────────────┬────────────────┘
             │
             ▼
        Final Answer
  (returned via FastAPI JSON)
```

**Why RAG?**  
RAG grounds the LLM's response in factual, domain-specific documents rather than relying solely on parametric knowledge. This reduces hallucinations and keeps answers accurate and relevant.

## 🚀 Quick Start

```bash
# 1. Create and activate virtual environment
python -m venv venv
source venv/bin/activate          # Windows: venv\Scripts\activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Generate the FAISS vector index
python backend/embed.py

# 4. Start the FastAPI server
uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000
```

## 🌐 API Usage

### Check server status
```
GET http://127.0.0.1:8000/
```

### Ask a gym question

## ⚙️ Changing the LLM

Open `backend/generator.py` and change `MODEL_NAME`:

```python
# Lightweight (default — runs on CPU)
MODEL_NAME = "TinyLlama/TinyLlama-1.1B-Chat-v1.0"

# Slightly larger, very capable
MODEL_NAME = "microsoft/phi-2"

# More powerful (requires 8GB+ RAM)
MODEL_NAME = "mistralai/Mistral-7B-Instruct-v0.2"
```
